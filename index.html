문자 작성

게스트 정보를 입력하면 템플릿에 자동으로 채워집니다

💬 문자 작성 사용 방법

  • ① 숙소/호실 — 보낼 대상 호실을 고르면 해당 와이파이·도어락이 자동 반영됩니다.
  • ② 게스트 정보 — 이름과 연락처 입력. 연락처는 자동 하이픈.
  • ③ 체크인/체크아웃 — 날짜 버튼을 눌러 입·퇴실을 한번에 선택. 인원·금액도 여기서.
  • ④ 템플릿 선택 — 단계별 템플릿을 고르면 우측 프리뷰에 실제 문자가 나타납니다.
  • {변수}는 {이름}, {입실시간}, {호실명}, {와이파이이름}, {결제금액} 등이 자동 치환됩니다.
  • 문자 앱 열기를 누르면 iOS/Android 문자 앱이 열리고 발송 이력에 자동 기록됩니다.
  • 예약 저장으로 이 예약을 "예약/발송 → 예약 목록"에 보관할 수 있어요.

템플릿 관리

단계별 문자 템플릿을 추가·편집·삭제합니다

📚 템플릿 관리 사용 방법

  • 6개 단계 — 문의·예약·입실 전·입실 당일·체크인·체크아웃별로 템플릿을 모아놨어요.
  • 추가 — 각 단계의 + 템플릿 추가를 눌러 새 문자를 만듭니다.
  • 편집 — 템플릿 제목 행을 클릭하면 펼쳐지고, 제목·본문을 수정 후 저장.
  • 복제 — 비슷한 문자를 빠르게 만들 때 사용 (해당 단계에 복사본 추가).
  • 사용 가능한 변수: {이름} {입실시간} {퇴실시간} {인원수} {결제금액} {연락처} {숙소명} {호실명} {숙소주소} {입금계좌} {와이파이이름} {와이파이비번} {도어락비번} {주중요금} {주말요금}
  • 변경 사항은 즉시 문자 작성에도 반영됩니다.

예약/발송

저장된 예약이 없습니다

📋 예약/발송 사용 방법

  • 예약 목록 탭 — 문자 작성 탭에서 예약 저장으로 보관한 게스트 예약을 관리합니다. 카드를 클릭하면 해당 예약이 문자 작성에 자동 로드돼요.
  • 발송 이력 탭 — 문자 앱 열기를 눌렀던 모든 발송 기록이 자동 저장됩니다.
  • 다시 보내기 — 이력의 문자 내용 그대로 문자 앱이 다시 열려요.
  • 복사 / 삭제 — 이력별 개별 조작, 전체 삭제로 이력 초기화도 가능.
  • 예약 카드의 오늘 입실·퇴실·예정 라벨로 상태를 한눈에 확인.

숙소 프로필

숙소 정보·호실·요금을 설정하면 문자에 자동으로 반영됩니다

🏠 숙소 프로필 사용 방법

  • 숙소 정보 — 이름·주소·계좌를 입력하면 {숙소명} {숙소주소} {입금계좌} 변수에 자동 반영됩니다.
  • 호실 관리 — 호실마다 와이파이·도어락을 따로 저장. 다중 호실 운영에 편리합니다.
  • 도어락 자동 — 손님 전화번호 뒷자리 자동 사용을 체크하면 게스트마다 도어락 비번이 자동 생성돼요.
  • 요금 — 주중·주말 요금을 저장하면 {주중요금} {주말요금}에 반영.
  • 내보내기 / 가져오기 — 설정·템플릿·예약 전체를 JSON 파일로 백업·복원할 수 있어요.
  • 홈 화면에 바로가기 추가 — 모바일 브라우저에서 누르면 PWA 앱처럼 설치됩니다.

🚀시작하기
이 앱은 숙소 예약 문자 템플릿을 빠르게 작성·발송하는 도구입니다.
게스트 정보를 입력하면 템플릿에 변수가 자동으로 채워집니다.
먼저 할 일
숙소 프로필 메뉴에서 숙소 이름, 주소, 계좌, 와이파이, 도어락, 요금을 입력하세요.
→ 문자 템플릿에 자동으로 채워집니다.
문자 작성 흐름
1단계 · 문자 작성 → 템플릿 선택 (문의/예약전/확정/입실전/입실후/퇴실)
2단계 · 게스트 정보 입력 (이름·연락처·날짜·인원·금액)
3단계 · 오른쪽 미리보기에서 확인 (SMS/LMS/MMS 자동 판별)
4단계 · 텍스트 복사 또는 문자 앱 열기로 발송
5단계 · 예약 저장으로 대시보드·예약 목록에 기록
🔤변수 목록
템플릿 편집 시 아래 변수를 사용하면 자동 치환됩니다.
🖋템플릿 편집
✏️ 템플릿 편집 버튼으로 제목과 내용을 직접 수정할 수 있어요.
· 변수 칩/이모지 칩을 탭하면 커서 위치에 삽입· 저장을 눌러야 최종 저장
＋ 새 템플릿 버튼으로 원하는 단계에 템플릿을 추가할 수 있어요.
기본 템플릿은 삭제 불가(수정만 가능). 직접 만든 템플릿은 삭제 가능.
📲여러 기기 동기화
숙소 프로필 → 내보내기로 JSON 파일 저장 후,
다른 기기에서 가져오기로 불러오면 설정·템플릿·예약이 그대로 옮겨집니다.
