📖 사용 설명서

🚀 시작하기
이 앱은 숙소 예약 단계별 문자 템플릿을 빠르게 만들어주는 도구예요.
손님 정보를 입력하면 템플릿에 자동으로 변수가 채워져요.
⚙️ 설정 먼저 하기
오른쪽 상단 ⚙️ 설정에서 숙소 정보를 입력하면
문자 템플릿에 자동으로 채워져요.
🏡 숙소 이름·주소·계좌 — 문자에 자동 입력
📶 와이파이·도어락 비번 — 입실 안내 문자에 자동 입력
🔑 전화번호 뒷자리 자동 사용 — 도어락 번호로 활용
💰 주중·주말 요금 — {주중요금} {주말요금} 변수에 반영
👤 손님 정보 입력
화면 상단 손님 정보 카드에 아래 항목을 입력하세요.
👤 이름 — 문자에 {이름}으로 자동 입력
📞 연락처 — 문자 보내기 시 자동 연결
📅 날짜 — 입실·퇴실 날짜, 박수 자동 계산
👥 인원 — {인원}에 자동 입력
💰 금액 — {금액}에 자동 입력
📋 단계별 탭
💬 문의 — 처음 문의 시 숙소 소개 문자
📋 예약전 — 날짜 확인 및 예약금 안내
✅ 예약확정 — 입금 확인 후 예약 완료 안내
🗺️ 입실전 — 전날·당일 위치 및 주차 안내
🏡 입실후 — 와이파이·편의시설 이용 안내
🙏 퇴실 — 감사 인사 및 후기 요청
🔄 새로고침
헤더의 🔄 버튼을 누르면 앱을 새로고침해요.
설정 저장 후 변수가 바로 반영되지 않을 때,
또는 화면이 이상할 때 눌러주세요.
📱 문자 보내기 · 복사
📋 복사 — 클립보드에 복사 (카카오톡 등)
📱 문자 — 손님 번호로 바로 문자 앱 열기
✏️ 수정 — 해당 템플릿 직접 편집 및 저장
· 수정한 내용은 자동으로 저장돼요
🔤 변수 목록
템플릿 수정 시 아래 변수를 사용하면 자동으로 채워져요.
{이름} {입실} {퇴실} {인원} {금액} {연락처} {숙소} {주소} {계좌} {와이파이} {와이파이비번} {비번} {전번뒷자리} {주중요금} {주말요금}
➕ 템플릿 수정·추가·삭제
✏️ 수정 버튼 → 제목과 내용을 직접 편집
· 변수 칩을 탭하면 커서 위치에 자동 삽입
· 이모티콘도 같은 방식으로 삽입 가능
· 저장 버튼을 눌러야 최종 저장돼요
➕ 템플릿 추가 버튼 → 각 단계에 새 템플릿 추가
· 단계별로 여러 개 만들 수 있어요
· 추가한 템플릿은 삭제 버튼이 표시돼요
⚠️ 기본 제공 템플릿은 삭제 불가, 수정만 가능해요
· 설정 → 초기화 시 수정 내용이 원래대로 돌아와요
📲 여러 기기 동기화
설정 → 📤 내보내기로 JSON 파일 저장 후
다른 기기에서 📥 가져오기로 불러오면
설정과 수정한 템플릿이 그대로 옮겨져요.
📲 앱으로 설치하기
🤖 안드로이드
Chrome 브라우저에서 접속 후
주소창 오른쪽 ⋮ 메뉴 → 홈 화면에 추가 선택
또는 하단에 설치 배너가 뜨면 설치 탭
🍎 아이폰 (iOS)
Safari 브라우저에서 접속 후
하단 공유 버튼(□↑) → 홈 화면에 추가 선택
· Chrome 등 다른 브라우저에서는 설치 불가
💻 컴퓨터 (PC·Mac)
Chrome 또는 Edge 브라우저에서 접속 후
주소창 오른쪽 설치(⊕) 아이콘 클릭
또는 ⋮ 메뉴 → 앱으로 설치 선택
· 설치 후 바탕화면에서 바로 실행 가능
