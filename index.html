📖 도움말

StaySMS 사용 방법과 팁을 알려드립니다

🚀시작하기
이 앱은 숙소 예약 문자 템플릿을 빠르게 작성·발송하는 도구입니다.
게스트 정보를 입력하면 템플릿에 변수가 자동으로 채워집니다.
⚙️먼저 할 일
숙소 프로필 메뉴에서 숙소 이름, 주소, 계좌, 와이파이, 도어락, 요금을 입력하세요.
→ 문자 템플릿에 자동으로 채워집니다.
✏️문자 작성 흐름
1단계 · 문자 작성 → 템플릿 선택 (문의/예약전/확정/입실전/입실후/퇴실)
2단계 · 게스트 정보 입력 (이름·연락처·날짜·인원·금액)
3단계 · 오른쪽 미리보기에서 확인 (SMS/LMS/MMS 자동 판별)
4단계 · 텍스트 복사 또는 문자 앱 열기로 발송
5단계 · 예약 저장으로 대시보드·예약 목록에 기록
🔤변수 목록
템플릿 편집 시 아래 변수를 사용하면 자동 치환됩니다.
🖋템플릿 편집
✏️ 템플릿 편집 버튼으로 제목과 내용을 직접 수정할 수 있어요.
· 변수 칩/이모지 칩을 탭하면 커서 위치에 삽입· 저장을 눌러야 최종 저장
＋ 새 템플릿 버튼으로 원하는 단계에 템플릿을 추가할 수 있어요.
기본 템플릿은 삭제 불가(수정만 가능). 직접 만든 템플릿은 삭제 가능.
📲여러 기기 동기화
숙소 프로필 → 내보내기로 JSON 파일 저장 후,
다른 기기에서 가져오기로 불러오면 설정·템플릿·예약이 그대로 옮겨집니다.
📱앱으로 설치
🤖 안드로이드 (Chrome): ⋮ 메뉴 → 홈 화면에 추가
🍎 아이폰 (Safari): 하단 공유(□↑) → 홈 화면에 추가
💻 PC (Chrome/Edge): 주소창 오른쪽 설치(⊕) 아이콘 클릭
