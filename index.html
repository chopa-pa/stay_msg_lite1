🚀 시작하기
이 앱은 숙소 예약 문자 템플릿을 빠르게 작성·발송하는 도구예요.
게스트 정보를 입력하면 템플릿에 변수가 자동으로 채워집니다.
⚙️ 먼저 할 일
숙소 탭에서 숙소 이름, 주소, 계좌, 와이파이, 도어락, 요금을 입력하세요.
→ 문자 템플릿에 자동으로 들어가요.
✏️ 문자 작성 흐름
1. 작성 탭에서 템플릿 선택 (문의/예약전/확정/입실전/입실후/퇴실)
2. 게스트 정보(이름·연락처·날짜·인원·금액) 입력
3. 오른쪽 미리보기에서 확인 (글자수·SMS/LMS 자동 판별)
4. 복사 또는 문자앱 버튼으로 발송
5. 예약 저장으로 홈 대시보드·예약 목록에 기록
🔤 변수 목록
템플릿 편집 시 아래 변수를 넣으면 자동 치환됩니다.
🖋 템플릿 편집
편집 버튼 → 제목과 내용을 직접 수정할 수 있어요.
· 변수 칩/이모지 칩을 탭하면 커서 위치에 삽입
· 저장을 눌러야 최종 저장
＋ 새 템플릿 추가 버튼으로 원하는 단계에 템플릿을 추가할 수 있어요.
기본 템플릿은 삭제 불가(수정만 가능). 직접 만든 템플릿은 삭제 가능.
🔄 새로고침
상단 🔄 버튼을 누르면 앱을 새로고침합니다.
설정 변경이 바로 반영되지 않을 때 사용하세요.
📲 여러 기기 동기화
숙소 탭 → 내보내기로 JSON 파일을 저장한 뒤,
다른 기기에서 가져오기로 불러오면 설정·템플릿·예약이 그대로 옮겨집니다.

🤖 안드로이드 (Chrome):
⋮ 메뉴 → 홈 화면에 추가
🍎 아이폰 (Safari):
하단 공유(□↑) → 홈 화면에 추가
💻 PC (Chrome/Edge):
주소창 오른쪽 설치(⊕) 아이콘 클릭
